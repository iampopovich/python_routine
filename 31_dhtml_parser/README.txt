1. необходимые зависимости : 
-python 3.6+
-selenium  3.141.0+
-cromedriver 80+
2. настройка конфига 
- в файл конфига cred.cfg ввести данные
-- login - логин на ХХ
-- password - пароль на ХХ
-- resume_id - ИД резюме на ХХ (адрес открытой страницы с резюме)
-- browser_path - путь до вебдрайвера (дефолтное значение chromedriver) # "browser_path":"chromedriver",
-- reply - кол-во ответов на вакансии в режиме автомата
-- reply_message - текст сопроводительного письма для ответа на вакансию
3. troubleshooting
- сейчас написано на коленке - путь к драйверу нужно прописать в PATH на Windows или закинуть файл драйвера в /usr/bin на linux-based осях
- позже ддобавлю ВЕНВ , чтоб не париться с настройкой окружения
- для запуска на macOS необходимо разрешить запуск хромдрайвера командой из корня каталога xattr -d com.apple.quarantine chromedriver
